Bakemono's Box

Rules

  • User has 3 guesses to guess the word letter by letter. If user failed to guess the word, the game will automatically restart
  • Each questions consists of 10 points
  • User has the ability to use a hint. They can choose what type of letter (vowel or consonant) they want to reveal.
  • Usage of hint is 25 points. User can use a hint if their points is less than 25 points
